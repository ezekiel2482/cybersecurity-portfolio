## Projects

###  [SIEM Log Forwarding & Centralized Logging](./siem-log-forwarding)
Configured a two-VM environment (Kali Linux as rsyslog server, Ubuntu as client) to forward system logs centrally, simulating a basic SIEM ingestion pipeline. Covers log source configuration, troubleshooting, and verification of forwarded events.

###  [Vulnerable VM Exploitation — Basic Pentesting 1](./vulnerable-vm-exploitation)
Full case study exploiting a deliberately vulnerable Linux VM: service enumeration with Nmap, vulnerability research, exploitation via Metasploit to gain root access, and password hash extraction/cracking. Demonstrates understanding of attacker methodology — knowledge that directly supports faster, more accurate alert triage on the defensive side.

###  [Incident Response Plan](./incident-response-planning)
A written incident response plan covering identification, containment, eradication, and recovery steps for a simulated security incident.

###  [Network Penetration Testing & OSINT](./network-penetration-testing)
Reconnaissance and information-gathering exercise covering domain reachability testing, subdomain enumeration, technology fingerprinting, and Google dork techniques.

###  [Digital Forensics — File Metadata Extraction](./digital-forensics)
A Python script (`forensics.py`) built using the `os` module to extract and display file metadata (timestamps, size, permissions) — a foundational step in establishing a timeline during a forensic investigation.
